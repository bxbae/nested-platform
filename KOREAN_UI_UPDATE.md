# Nested 메인 UI·한국어 콘텐츠 개편

반영 내용
- 히어로 배경에 약한 블러와 좌측 가독성 그라데이션 적용
- 히어로 검색창을 전체 폭 중앙 정렬
- 상단의 중복 서비스 카드 제거
- `세 번의 검증, 한 번의 입주`를 히어로 바로 아래로 이동
- `어떤 집을 찾으세요?` 카테고리 섹션 제거
- 숙소명·지역명·소개·편의시설·커뮤니티 목업을 한국어화
- Prisma 시드 숙소명과 지역을 한국어화
- 기존 영어 시드 숙소도 재시드 시 함께 제거하도록 legacyName 처리

적용 후 확인
1. 프론트: `npx tsc --noEmit && npm run build`
2. 백엔드: `npx prisma generate && npm run build`
3. DB 시드 반영이 필요하면 백업 후 `npx prisma db seed`
